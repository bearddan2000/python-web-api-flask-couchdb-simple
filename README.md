# python-web-api-flask-couchdbdb-simple

## Description
Creates an api of `dog` document.
Has the ability to query by parameters.

Remotely tested with *testify*.

## Tech stack
- python
  - flask
  - couchdb
  - testify
  - requests

## Docker stack
- python:latest
- couchdb:latest

## To run
`sudo ./install.sh -u`
- Get all routes: http://localhost/help

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
